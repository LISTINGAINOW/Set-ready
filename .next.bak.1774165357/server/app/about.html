About Discreet Set

Private locations for private productions.

Discreet Set is a location marketplace built for productions that need professionalism, privacy, and speed. Instead of forcing adult-friendly shoots into platforms that create friction, the business is designed around the real workflow from day one.

What is Discreet Set?

It’s a curated marketplace for homes, studios, lofts, warehouses, cabins, and other filmable properties that are open to private productions. Listings are presented with privacy controls, transparent booking context, and a cleaner production-first experience than generic rental platforms.

Mission

Private locations for private productions.

The mission is simple: help producers book the right set faster while giving hosts the control they need to feel safe saying yes.

Team

Lean team. Clear standards.

Discreet Set is built by operators focused on privacy-sensitive productions, marketplace quality, and real-world host relationships. The goal is not to be noisy. The goal is to be reliable.

That means better listings, stronger host vetting, and a tone that respects both the business reality and the discretion the category demands.
